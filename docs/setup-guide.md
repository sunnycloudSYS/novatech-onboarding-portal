# NovaTech — New Hire Setup Guide

Welcome to NovaTech! Follow these steps to get your development environment ready.

## Step 1 — Accounts
- [ ] Create your GitHub account (use a professional username)
- [ ] Request access to the NovaTech GitHub organization (ask Alex)
- [ ] Set up your Slack profile with your real name and role

## Step 2 — Tools
- [ ] Install Git on your machine
- [ ] Configure Git with your name and NovaTech email
- [ ] Install VS Code (or your preferred editor)
- [ ] Install a terminal (Git Bash on Windows, default Terminal on Mac/Linux)

## Step 3 — Verify
- [ ] Run git version check to confirm Git is installed
- [ ] Run git config check to confirm your identity is set
- [ ] Clone any NovaTech repo to confirm GitHub access works

## Step 4 — First Day Checklist
- [ ] Read the Team Directory
- [ ] Complete this Setup Guide
- [ ] Read the Company FAQ
- [ ] Introduce yourself in #general on Slack

You're ready. Welcome to the team!
