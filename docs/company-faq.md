# NovaTech — New Hire FAQ

## General

**Q: What's the dress code?**
A: There isn't one. Wear what makes you productive.

**Q: What are the working hours?**
A: Core hours are 10am - 4pm WAT. Outside of that, work when you're most productive.

**Q: Who do I ask if I'm stuck?**
A: Your team lead (Alex) first. If Alex is unavailable, post in #engineering-help on Slack.

## Git & GitHub

**Q: Should I commit directly to main?**
A: Never. Always create a feature branch and open a pull request.

**Q: How often should I commit?**
A: Commit small, commit often. Each commit should do one thing and have a clear message.

**Q: What if I accidentally commit a secret (like an API key)?**
A: Tell your team lead immediately. The key needs to be rotated even if you delete the commit — Git history is permanent.

## Tools

**Q: What editor should I use?**
A: Most of the team uses VS Code, but use whatever you're comfortable with.

**Q: Do I need a Linux machine?**
A: Not required for onboarding, but you'll need SSH access to servers later. We'll set that up in your second week.
